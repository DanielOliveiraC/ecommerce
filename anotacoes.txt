Projeto Ecomerce - Java & Spring Boot (FULL STACK)

Proposta: 
 Desenvolver uma aplicação full stack para gerenciamento de um supermercado/e-commerce, implementando uma API RESTful com Java e Spring Boot. 
 O objetivo é demonstrar boas práticas de desenvolvimento, documentação de APIs e integração com banco de dados NoSQL (MongoDB), publicando o
 projeto no GitHub e compartilhando no LinkedIn.

Requisitos funcionais:
 - CRUD Clientes
 - CRUD Produtos 
 - Gestão de Pedidos 
 - Gerenciar Estoque 
 - Persistência de dados em banco NoSQL (MongoDB)
 - Geração de relatórios sobre vendas

Requisitos não funcionais:
 - Integração com Swagger para documentação
 - Código respeitando conceitos SOLID 
 - Containerização com Docker
 - Documentação clara no README

Tecnologias:
    Backend:
        - Java
        - Spring Boot
        - Spring Data MongoDB
        - (A definir se terá autenticação por token)
    
    Batabase:
        - MongoDB

    Documentação:
        - Swagger / OpenAPI

    Ferramentas:
        - Maven
        - Docker

    Frontend: 
        - A definir
